# TOPIK 歌词词汇与语法学习软件

一个无需后端的学习应用，已支持 PWA，可在 iPhone 上“添加到主屏幕”作为 App 使用。

## 功能
- 韩语歌曲例句学习 TOPIK 词汇和语法
- 学习界面可直接点击歌曲链接试听
- 每个条目可设置掌握程度（0-3）
- 根据复习正确率与掌握程度推荐：
  - 未学习内容优先
  - 低正确率内容优先
  - 其余内容均衡复习
- 学习状态自动保存在浏览器 `localStorage`

## 使用
1. 直接双击打开 `index.html`，或用任意静态服务器打开。
2. 在页面中点击“我答对了 / 我答错了”进行复习。
3. 点击“点开听”可跳转到歌曲链接。

## iPhone 当 App 使用
1. 先把项目部署成可访问网址（推荐 GitHub Pages / Vercel / Netlify）。
2. 在 iPhone 的 Safari 打开你的网址。
3. 点击底部分享按钮，选择“添加到主屏幕”。
4. 回到主屏后，从图标进入即为全屏 App 体验。

注意：iPhone 不能把本地 `index.html` 直接安装为主屏 App，必须通过网页地址访问。

## 自定义数据
编辑 `app.js` 的 `learningItems` 数组即可增加/替换词汇、语法、例句和歌曲链接。
